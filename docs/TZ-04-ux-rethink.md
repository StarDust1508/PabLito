# ТЗ-04 (МАСТЕР): UX-переосмысление + режимы-треды + светлая тема

**Единый документ для агента.** Включает в себя ТЗ-03 (режимы/светлая тема/острова/полировка) и решения по 4 развилкам. ТЗ-02 уже собран — его фичи (перевод, фото, поиск, кинетический логотип, шрифты, шторка) сохраняем, но реорганизуем по новой информационной архитектуре ниже. Этот документ **заменяет ТЗ-03**.

Код качественный, но опыт построен неверно: «дашборд с кнопкой на каждую фичу» вместо «разговор с живым другом». Чиним смысл и удобство.

---

## 0. Диагноз (почему «неудобно»)
1. **Чат — не дом.** Перед лентой 4–5 полос чрома (шапка, стрик+📖+🧠, РЕЖИМ, города). Главная функция зажата внизу.
2. **Функция = кнопка сверху.** Práctica, Слова, Обо мне, стрик, счётчик, города, ánimo, режим — всё на верху. Много места, шум, нет иерархии.
3. **Разрыв код↔смысл.** Фичи «кликаются», а не «ощущаются»: память живёт в 🧠 вместо реплик; произношение — модалка вместо инлайн-оценки; настроение — плашка вместо тона.

**Вывод:** проблема в информационной архитектуре и в том, что смысл функций не вплетён в опыт.

---

## 1. Принцип: разговор — дом, инструменты — по требованию
Три слоя:
- **Всегда видно:** минимальная шапка (аватар · имя · эмодзи-настроение · presence) + лента + ввод.
- **В один тап:** всё остальное — за ОДНОЙ боковой панелью (см. §6.2).
- **В потоке:** настроение — тон+эмодзи; память — сам вспоминает; коррекция — инлайн; активности — предлагает репликами/чипами.

---

## 2. Консолидация кнопок: «откуда → куда»
| Сейчас (полоса/кнопка сверху) | Куда |
|---|---|
| 🎙 Práctica | Панель → «Инструменты» + Паблито предлагает в потоке |
| 📖 Слова | Панель → «Прогресс» |
| 🧠 Обо мне | Панель → «Профиль» |
| 🔥 стрик + счётчик | Панель → «Прогресс» (в шапке максимум крошечный 🔥N по тапу) |
| Города/погода | Панель → «Инструменты»; на главном — по тумблеру (§6.1) |
| Плашка ÁNIMO | Эмодзи рядом с именем |
| Полоса «РЕЖИМ» | 2 треда + таблетка (§7) |
**Итог верха:** имя + статус + одна кнопка-меню. Постоянных фиче-кнопок — ноль.

---

## 3. Минимальная шапка (всегда)
Слева: аватар (тап → панель) · имя PabLito · эмодзи-настроение · presence. Справа: одна иконка-меню (или тап по аватару / свайп от левого края). Больше постоянного — ничего.

---

## 4. Смысл «в потоке», а не «в кнопках»
- **Память:** Паблито проактивно ссылается на факты/моменты; периодически «¿te acordás que…?». Экран 🧠 остаётся вторичным.
- **Коррекция:** инлайн-рекаст + опционально «разбор ошибок» в конце сессии одной репликой.
- **Произношение:** после голосового — мягкая инлайн-оценка в ответе; полноценная тренировка — в «Инструментах».
- **Настроение:** тон + эмодзи; плашку убрать.
- **Цель/переезд:** иногда связывает материал с жизнью в Аргентине; счётчик — в панели.

---

## 5. Чего не хватало (пробелы опыта)
1. **«Сегодня» / дневной импульс** — см. §6.4 (реализуем).
2. **Онбординг функций в разговоре** — первые дни Паблито ненавязчиво показывает возможности репликами.
3. **Голос на первом плане** — см. §6.3 (реализуем).
4. **Обзор роста речи** — раздел «над чем работаем» в панели (из `error_patterns`).
5. **Пустые состояния/подписи** — ничего немого.

---

## 6. РЕШЕНИЯ (вшиты по твоим ответам)

### 6.1 Города/погода — оставляем + тумблер «убрать»
- По умолчанию показываются (тебе нужны: время в Саратове/БА/Москве).
- В **Настройках** — тумблер «Показывать города на главном». Выкл → они уходят с главного, но доступны в панели → «Инструменты».
- На главном — компактнее (одна лёгкая строка/карточка), и скрываются при письме/раскрытии чата.

### 6.2 Навигация — одна боковая панель (как ChatGPT)
- Открытие: свайп от левого края **или** тап по аватару/иконке-меню.
- Никаких нижних таб-баров и топ-кнопок. Всё — здесь.
- **Секции панели:**
  - **Профиль:** имя, уровень, цель + счётчик до Аргентины.
  - **Прогресс:** стрик, слова 📖, вехи, «над чем работаем».
  - **Инструменты:** Произношение, Города/погода.
  - **История чатов:** 30 дней, метки режима (бело/синие) — просмотр, НЕ переключатель.
  - **Настройки:** тема (свет/тьма/системная), звук озвучки, пуш «пишет первым», тумблер городов (§6.1), «Обо мне»/стереть память.

### 6.3 Голос — ОСНОВНОЙ ввод
- Крупная круглая кнопка микрофона справа в строке ввода (≈54px, акцент). Паттерн **«удерживай — говори»**: зажал → запись (индикатор/уровень), отпустил → распознавание (whisper) → отправка.
- Текст — вторичный: тап по полю «Escribí…» открывает клавиатуру; при наборе появляется кнопка отправки, микрофон уступает место.
- Подпись-подсказка под лентой: «удерживай микрофон и говори — я слушаю».
- После голосового — Паблито даёт мягкую инлайн-оценку произношения (§4).

### 6.4 «Сегодня» — Паблито предлагает при входе (и разговор, и занятие)
- При открытии/новой сессии, после приветствия, Паблито даёт **2–3 чипа-предложения** (тап по чипу = отправляет это как реплику/запускает активность):
  - **Режим «Друг»:** разговорные заходы — «продолжить прошлую тему», «расскажи, как прошёл день», «слово дня», «поболтаем о …».
  - **Режим «Урок»:** учебные — «тема дня: кафе», «повторить слова», «2 мин произношения».
- Мягко, не навязчиво; чипы можно проигнорировать и просто писать/говорить.
- Тон приветствия — по настроению/близости/паузе (уже есть).

---

## 7. Режимы = ДВА ОТДЕЛЬНЫХ ЧАТА (из ТЗ-03, критично)
- **Проблема:** сейчас Друг и Урок — один чат с разным промптом → сообщения наслаиваются; переключатель в шторке путал (принимали за сортировку).
- **Надо:** Друг и Урок — **два независимых треда**, у каждого своя история. Переключение = показать другой тред целиком (не подмешивать). Настроение/память/профиль — общие (один Паблито).
- **Индикатор:** полупрозрачная **таблетка** «💬 Друг» / «📚 Урок» вверху ленты; тап по ней или по чистому сегменту-островку — сменить тред.
- Убрать слово «РЕЖИМ» и переключатель режима из шторки (там — только история).
- **Где:** `usePablito.ts` (тред/история по `mode`, resume по режиму), `ChatScreen.tsx` (таблетка), `ProfileDrawer.tsx` (убрать переключатель).

---

## 8. Светлая тема + переключатель (из ТЗ-03)
- Настройки → тема: **Светлая / Тёмная / Системная** (сохранять выбор; сейчас берётся системная).
- Довести **светлую** до malvah-вида: тёплый бумажный фон (`#EDEAE3`/`#F5F3EE`), почти чёрный текст (`#141210`), celeste-акцент (`#3E6DDb`), тонкие линии, мягкие тени вместо жирных рамок. Ориентир — присланный макет.

---

## 9. Ввод-остров (из ТЗ-03 + голос-основной §6.3)
- Строка ввода: **[малый полупрозрачный островок: 📎 скрепка]** · **[поле текста]** · **[крупный акцентный микрофон]**.
- Скрепка (и доп. действия) — в компактном полупрозрачном островке. Микрофон — крупный, основной (§6.3). Текст по центру поля (уже поправлено). Иконки — lucide.

---

## 10. Полировка (из ТЗ-03 — почему «выглядит плохо»)
- **Пузыри:** убрать моно-лейблы `VOS`/`PABLITO`; реплики Паблито — с маленьким аватаром слева, user — справа; мягкие тени вместо жирных рамок; консистентные радиусы.
- **Верх облегчить**, единая сетка 8pt, больше воздуха.
- **Кинетический логотип** — приглушить амплитуду (лёгкое «дыхание»).
- **Иконки** — только lucide, единая толщина.

## 12. Принципы (чтобы не отрастить кнопки заново)
- **Новая функция ≠ новая кнопка сверху.** Сначала: можно ли отдать это Паблито в поток? Если нет — в панель.
- **Главный экран — заповедник**: постоянного чрома минимум.
- **Смысл важнее наличия**: фича сделана, только если ощущается в опыте.

## 13. Приёмка (единый чек-лист)
- [ ] Верх минимальный: имя, эмодзи-настроение, presence, одна кнопка-меню. Ни Práctica, ни ánimo-плашки, ни россыпи иконок.
- [ ] Панель (свайп слева/тап аватара): Профиль · Прогресс · Инструменты · История · Настройки.
- [ ] В Настройках: тема (свет/тьма/системная), звук, пуш, **тумблер городов**, стереть память.
- [ ] Города по умолчанию на главном (компактно), тумблером убираются в панель.
- [ ] Голос — основной: «удерживай — говори»; текст по тапу; после голосового — инлайн-оценка.
- [ ] При входе Паблито предлагает 2–3 чипа: в «Друге» — разговорные, в «Уроке» — учебные.
- [ ] Режимы — два отдельных треда, ничего не наслаивается; таблетка режима на чате; в шторке переключателя режима нет.
- [ ] Светлая тема выглядит премиально; пузыри без VOS/PABLITO, с аватаром и мягкими тенями; логотип приглушён.
